# SkinSync

Overview
SkinSync is a community-driven skincare tracking app that empowers users to document their daily skincare routines. It promotes healthy habits and fosters motivation through social interaction and accountability. Users can build and maintain daily Skincare Streaks—similar to Snapchat streaks—by completing their routine and uploading a photo each day.

Key Features
- Users can add their own custom daily skincare routines
- Record and track daily skincare routines
- Maintain streaks to build consistent habits
- Upload daily progress pictures
- Engage with a community that supports and motivates each other

Tech Stack & Dependencies

- Firebase Auth – User authentication
- Cloud Firestore – Real-time database
- Supabase Storage – Store routine images
- Image Picker – Capture and upload photos
- FL Chart – Visualize progress and trends
- State Management – Powered by GetX
